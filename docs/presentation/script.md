## 2. 데모 — 데스크탑 (150초)
> 발표 5분 전 사이트 한 번 열어 **서버 예열**(콜드스타트 방지). 로그인된 상태로 시작.

**① 메인 — 정체성** `[홈]`
"좌측 상단에 **'신화 부동산 쇼핑몰 · 공인중개사를 위한 올인원 큐레이션 마켓'** — 누가 봐도 **뭘 하는 곳인지** 바로 보이게 했어요.
핵심은 **'큐레이션'** 입니다. 아무거나 파는 게 아니라, **공인중개사 현장에 진짜 필요한 것만** 골라 담았어요."

**② 카테고리 — 중개사 맞춤 4종** `[카테고리 클릭]`
"**사무용품 · 교육콘텐츠 · 경매 컨설팅 · AI 과정·코칭** — 전부 중개사 일과 직결된 카테고리입니다. 누르면 해당 상품만 걸러져요."

**③ 상품 상세 — 이 쇼핑몰의 '시그니처'** `[상품 클릭]`
"제가 제일 신경 쓴 특징인데요. 상품을 **실물**과 **디지털·서비스**로 나눴습니다. 성격이 다르니까 **사는 방식도 다르게** 했어요."
- 실물(명함·도장·현수막): "배송비가 붙고 **수량을 조절**합니다."
- 디지털(VOD·AI코칭): "**무료·수량 1개 고정**, 이메일/일정으로 즉시 이용. VOD를 2개 살 일은 없으니까요."

**④ 장바구니 → 결제 → 주문완료** `[담기 → 주문하기 → 포트원 결제]`
"장바구니에서 금액·배송비가 **실시간 계산**되고요. 결제는 **포트원**으로 실제 결제창이 뜹니다. 그리고 **결제가 확인된 뒤에만 주문이 생성**되도록 했어요 — 유령 주문을 막으려고요."
`[주문완료]` "주문번호와 다음 단계 안내까지 나옵니다."

**⑤ 관리자 — 운영까지** `[admin → /admin]` *(빠르게)*
"운영자 화면도 있어요. **대시보드에 매출·주문 통계**, **주문 상태 변경**(제작중·배송중·완료), **상품 등록·수정·삭제**까지 — 실제로 가게를 운영할 수 있게요."
